Welcome to your new dbt project!

### Run Project Manually

Try running the following commands:
- dbt run
- dbt test

## Run Project Using Airflow

- install astro cli: https://www.astronomer.io/docs/astro/cli/install-cli/
- cd dbt-dag
- astro dev start
